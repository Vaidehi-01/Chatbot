# BOOK RECOMMENDATION CHATBOT

### create a virtual env
```
python3 -m venv .venv

or

python -m venv .venv

```
### start the  virtual environment by
```
# in windows

.\.venv\Scripts\activate

# in linux

./venv/bin/activate
```
### install the requirements
```
pip install -r requirements.txt
```
### run the application
```
flask run
```
